First public release of `steam`, a single-binary CLI for the Steam game store.
No API key required.

## Commands

- `steam top-sellers` — top-selling games right now, `-n` to limit
- `steam new-releases` — newest released games, `-n` to limit
- `steam specials` — games currently on sale (discount > 0), `-n` to limit
- `steam search <query>` — search the store catalog, `-n` to limit
- `steam game <id>` — full details for one game by app ID or store URL
- `steam reviews <id>` — recent user reviews for a game, `-n` to limit

## Output formats

All list commands support `-o table|json|jsonl|csv|tsv|url`. The default is
`table` on a TTY and `jsonl` when piped.

## Install

```bash
# Homebrew
brew install tamnd/tap/steam

# Go install
go install github.com/tamnd/steam-cli/cmd/steam@latest

# Docker
docker run --rm ghcr.io/tamnd/steam:0.1.0 top-sellers
```
