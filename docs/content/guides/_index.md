Each guide is built around a job rather than a command. They assume you have run
the [quick start](/getting-started/quick-start/).

- [Resource URIs](/guides/resource-uris/) is how you use st as a
  `database/sql`-style driver so a host can address Steam as `steam://` URIs and
  walk the graph.
- [Add a command](/guides/adding-a-command/) is how you model a new Steam record
  and turn it into a command, a route, and a tool at once.
